Profiling de Cada Plataforma

Se você está portando este código para outra plataforma e está fazendo o profiling de suas alterações que compartilha-las com a comunidade.

Eu criei uma pasta no repositório chamada ""profiling", para vela clique no link e veja detalhes no arquivo Readme.md.

Para cada plataforma haverá um diretório com pelo menos um arquivo Readme.md com instruções de ocmo efetuar o profiling.

Por hora eu tenho trabalho na plataforma NanoPI M3, em breve irei levar a experiência para o RaspberryPI, apesar que achar o NanoPI muito superior e muito adequado a este projeto, mas pretendo fazerum pequeno comparativo de desempenho. O NanoPI M3, usa um Cortex-A53 que tem um excelente desempenho. São 8 cores rodando entre 400MHz e 1.4GHz.
